# Input Format

 ```
  n m
  s t
  u[1] v[1] w[1]
  ...
  ...
  u[m] v[m] w[m]
```
 - There will be total m + 2 lines input
 - Notations are,
 	- n: number of vertices
 	- m: number of edges
 	- s: source vertex
 	- t: sink vertex
 	- u[i]: vertex u, 1 <= u <= n
 	- v[i]: vertex v, 1 <= v <= n
 	- w[i]: capacity of ith edge
 	- ith edge: u -> v directed edge
